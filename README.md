## Installation

```sh

# Clone the repo
$ git clone https://github.com/Ryuk-me/pocket-url

# Move to folder
$ cd pocket-url

# Install PNPM
$ npm install -g pnpm

# Install Packages
$ pnpm install

# Build Projetc
$ pnpm run build

# Run 
$ pnpm run dev

>> Using docker

# Build
$ docker-compose build

# Start
$ docker-compose up

```
